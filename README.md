## ✨ Features

- 🔍 Single Website Search
- 📋 Multiple Website Search
- 📂 Upload Text File
- 🌍 Interactive World Map
- 📍 Colored Markers
- 🔥 Heat Map
- 🥧 Pie Chart
- 📄 Server Information Table
- ⬇️ Download CSV

---

## 🛠 Tech Stack

- Python
- Streamlit
- Pandas
- Folium
- Requests
- Matplotlib

---

## 🚀 Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
